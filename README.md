# C++ 学习之路 (基于黑马程序员视频教程)

本项目用于记录和归档在学习 **黑马程序员C++视频教程** 过程中的代码练习与学习笔记。

## 🛠️ 开发环境
* **IDE**: Microsoft Visual Studio (版本 17 / VS2022)
* **项目配置**: C++ 控制台应用程序 (Console)
* **平台架构**: 同时支持 Win32 (x86) 和 x64 平台编译

## 📂 项目结构与状态
目前项目已经完成了基础的初始化，包含以下核心文件：

* `write  helloworld.cpp`: 学习的第一行代码，演示了 C++ 程序的标准结构（引入 `<iostream>`、使用标准命名空间、`main` 函数）、单行与多行注释的使用，以及使用 `cout` 输出“hello world”。
* `c++studing.sln` / `.vcxproj`: Visual Studio 的解决方案与项目工程文件。
* `.gitignore` & `.gitattributes`: 已配置好针对 Visual Studio 常见临时文件和编译生成文件的 Git 忽略规则，以及跨平台文本换行符的自动处理。
